# Görev 5 - Fikirler ve Öneriler

## 1. Yeni Liste Oluşturma İyileştirmeleri

### 1.1 Modal ile Yeni Liste
- İsim, başlangıç görevleri, renk seçimi içeren modal
- Şablon listeler (Alışveriş, Proje, Günlük rutin)
- Son kullanılan listelerden kopyalama

### 1.2 Hızlı Oluşturma
- Sürükle-bırak ile kartları yeniden sıralama
- Görevleri bir karttan diğerine taşıma
- Toplu görev ekleme (çok satırlı yapıştırma)

## 2. Kart Özellikleri

### 2.1 Görsel Özelleştirme
- Kart rengi seçimi
- Etiket/kategori sistemi
- Öncelik seviyesi (düşük, orta, yüksek)

### 2.2 İlerleme Göstergesi
- Progress bar (tamamlanan görev yüzdesi)
- Kalan görev sayısı badge'i
- Tamamlanma animasyonu

## 3. Görev Özellikleri

### 3.1 Alt Görevler
- İç içe görevler (nested todos)
- Görev grupları
- Bağımlılık tanımlama

### 3.2 Tarih ve Hatırlatma
- Bitiş tarihi ekleme
- Hatırlatma ile entegrasyon
- Tekrarlayan görevler

### 3.3 Notlar
- Görev açıklaması
- Dosya eki
- Link ekleme

## 4. Filtreleme ve Arama

### 4.1 Filtreler
- Tamamlananları göster/gizle
- Tarihe göre sırala
- Önceliğe göre sırala

### 4.2 Arama
- Görev metni arama
- Etiket bazlı arama
- Tarih aralığı filtreleme

## 5. Senkronizasyon

### 5.1 Export/Import
- JSON formatında export
- Markdown formatında export
- Başka uygulamalardan import

### 5.2 Bulut Senkronizasyon
- Google Tasks entegrasyonu
- Microsoft To-Do entegrasyonu
- Todoist API desteği

## 6. Gamification

- Tamamlanan görev sayacı
- Seri (streak) takibi
- Başarı rozetleri
